## Real Estate Helper is an AI-driven API that:

    Scrapes WhatsApp chats and extracts structured real-estate units list
    Stores unit details in database(postgresql)
    Allows users to ask natural-language questions (Arabic or English)
    Converts questions → SQL queries → getting friendly responses
    Runs as a clean Dockerized FastAPI service
